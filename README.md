# Project1-Msc-

# Reanalysis of the Of Stress Granule proteome to understand Neurodegeneration

This repository contains my R-based re-analysis of the proteomics data associated with the study: 
"Context-Dependent and Disease-Specific Diversity in Protein Interactions within Stress Granules"** (Markmiller et al., Cell 2018).

## Data Source
The dataset used in this project is sourced from MassIVE(ID: MSV000081554).
- [MassIVE Dataset Link](https://massive.ucsd.edu/ProteoSAFe/dataset.jsp?task=9c99413cc1fd46549629d4cc9c334589)

## Project Goals
- Re-analyze the APEX2 proximity labeling data from 293T cells and iPSC-derived neurons.
- Visualize the protein interaction networks within stress granules under different conditions.
- Compare findings with current literature on neurodegenerative disease markers.

## Tools Used
- R / RStudio
- MSFragger(for initial peptide identification)
- ggplot2(for data visualization)
